# Project_Aurora
Данный проект — это приложение для Sailfish OS, разработанное с использованием Aurora OS SDK и QtQuick. Он предназначен для отображения информации о расписании, включая детали уроков, преподавателей и аудиторий.

### Общая структура и компоненты

Проект представляет собой приложение для Sailfish OS/ Aurora OS, использующее QtQuick и Aurora SDK. Он включает main.cpp, несколько QML-страниц, компоненты интерфейса и модуль для работы с расписанием и базой данных.

### Общие рекомендации по использованию и расширению

Проект использует Theme — убедитесь, что он определен и содержит нужные переменные (размеры, цвета).
Для работы с расписанием желательно реализовать динамическую подгрузку данных, например, при смене ID группы.
Для расширения можно добавить новые виды карточек, фильтры или поддержку новых источников данных.
Важно своевременно обновлять базу данных и обрабатывать ошибки сетевых запросов.


### Итоги

Этот проект — полнофункциональное приложение, позволяющее выводить, сохранять и загружать расписание занятий, интегрированное с базой данных и API. Использование Aurora SDK и Sailfish.Silica обеспечивает современный интерфейс для устройств на Sailfish OS/Aurora OS.
